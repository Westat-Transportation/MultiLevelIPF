Estimate weights using Iterative Proportional Updating

Usage

ml_fit_ipu(fitting_problem, tol = 1e-06, maxiter = 500, verbose = FALSE)

Arguments

fitting_problem
A fitting problem created by fitting_problem.
tol
Tolerance, the algorithm stops when all target values are reached within this tolerance.
maxiter
Maximum number of iterations.
verbose
If TRUE, print diagnostic output.

Value

An object of classes ml_fit_ipu and ml_fit.

Description

This function reweights a reference sample to match constraints given by aggregate controls by means of Iterative Proportional Updating.

References

Ye, X., K. Konduri, R. M. Pendyala, B. Sana and P. A. Waddell (2009) A methodology to match distributions of both household and person attributes in the generation of synthetic populations, paper presented at the 88th Annual Meeting of the Transportation Research Board, Washington, D.C., January 2009.

Examples

path <- toy_example("minitoy")
ml_fit_ipu(fitting_problem = readRDS(path))
