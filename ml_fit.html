Estimate weights using a given algorithm

Usage

ml_fit(algorithm, ref_sample, controls, field_names, verbose, ...)

Arguments

algorithm
Algorithm to use
ref_sample
Reference sample. Alternatively, a configuration as returned by import_IPAF_results; in this case, the other arguments are ignored.
controls
Control totals
field_names
Names of special columns in the data
verbose
If TRUE, print diagnostic output.
...
Further parameters passed to the algorithm

Value

An object of class ml_fit, essentially a named list.

Description

This function reweights a reference sample to match constraints given by aggregate controls using an algorithm given as parameter.

Examples

path <- system.file("extdata/minitoy", package="MultiLevelIPF")
ml_fit(algorithm = "entropy_o", ref_sample = import_IPAF_results(path))
