Imports results from a run of the original Python implementation

Usage

import_IPAF_results(path, all_weights = FALSE, config_name = "config.xml")

Arguments

path
Path to the directory that contains the input/output files
all_weights
Should all weights be loaded, or only the final ones (default)?
config_name
Name of the main configuration file, defaults to "config.xml"

Value

An object of class IPAF_result, essentially a named list with the following components:
refSample
The reference sample, a data.frame.

controls
A named list with two components, individual and group. Each contains a list of controls as data.frames.

fieldNames
A named list with the names of special fields.

algorithms
A list of algorithm names.

weights
A named list with weight vectors, one per algorithm.

Description

This helper function imports input, data and results from a run of the original Python implementation into a named list. The file `config.xml` is read and interpreted.
