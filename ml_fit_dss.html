Estimate weights using generalized raking

Usage

ml_fit_dss(fitting_problem, method = c("raking", "linear", "logit"), ginv = MASS::ginv, verbose = FALSE)

Arguments

fitting_problem
A fitting problem created by fitting_problem.
method
Calibration method, one of "raking" (default), "linear", or "logit"
ginv
Function that computes the Moore-Penrose pseudoinverse, currently only the default value MASS::ginv is accepted.
verbose
If TRUE, print diagnostic output.

Value

An object of classes ml_fit_dss and ml_fit, essentially a named list.

Description

This function reweights a reference sample to match constraints given by aggregate controls by means of generalized raking.

Details

Internally, sampling::calib is called.

References

Deville, J.-C. and Särndal, C.-E. (1992) Calibration estimators in survey sampling. Journal of the American Statistical Association, 87(418), 376--382.

Deville, J.-C., Särndal, C.-E. and Sautory, O. (1993) Generalized raking procedures in survey sampling. Journal of the American Statistical Association, 88(423), 1013--1020.

path <- toy_example("minitoy")
ml_fit_dss(fitting_problem = readRDS(path))
