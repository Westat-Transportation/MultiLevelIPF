Estimate weights using Hierarchical Iterative Proportional Fitting

Usage

ml_fit_hipf(fitting_problem, tol = 1e-06, maxiter = 500, verbose = FALSE)

Arguments

fitting_problem
A fitting problem created by fitting_problem.
tol
Tolerance, the algorithm stops when all target values are reached within this tolerance.
maxiter
Maximum number of iterations.
verbose
If TRUE, print diagnostic output.

Value

An object of classes ml_fit_hipf and ml_fit.

Description

This function reweights a reference sample to match constraints given by aggregate controls by means of Hierarchical Iterative Proportional Fitting.
