Estimate weights using an entropy optimization approach

Usage

ml_fit_entropy_o(fitting_problem, verbose = FALSE, dfsane_args = list())

Arguments

fitting_problem
A fitting problem created by fitting_problem or import_IPAF_results.
verbose
If TRUE, print diagnostic output.
dfsane_args
Additional arguments (as a named list) passed to the dfsane function used internally for the optimization.

Value

An object of classes ml_fit_entropy_o and ml_fit, essentially a named list.

Description

This function reweights a reference sample to match constraints given by aggregate controls using an entropy optimization method.

Examples

path <- system.file("extdata/minitoy", package="MultiLevelIPF")
ml_fit_entropy_o(fitting_problem = import_IPAF_results(path))
